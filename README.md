# Data Mining & Machine Learning Projects — UFRJ

This repository contains 6 Data Mining and Machine Learning projects completed at the Federal University of Rio de Janeiro (UFRJ) during my exchange semester (2025.1). Each project is presented as a Jupyter notebook.

**Romain Sebire** — 125 009 460

---

## Projects

| Project | Description | Notebook |
|---------|-------------|----------|
| **Bus Location Prediction** | GPS data pipeline for Rio de Janeiro's bus network: depot/terminal detection, route construction, segment speed modeling | [View notebook](notebooks/en/Notebook_bus_en.ipynb) |
| **Store Sales Forecasting** | Time series prediction of daily store sales using holidays, oil prices, and payday features (Kaggle) | [View notebook](notebooks/en/Notebook_forecasting_en.ipynb) |
| **Clustering on MovieLens 1M** | User segmentation on the MovieLens 1M dataset using KMeans, DBSCAN, and Agglomerative clustering | [View notebook](notebooks/en/Notebook_clustering_en.ipynb) |
| **Credit Approval** | Classification model for credit default prediction with LightGBM (Kaggle competition) | [View notebook](notebooks/en/Notebook_creditApproval_en.ipynb) |
| **Obesity Risk Classification** | Multi-class classification of obesity risk levels using XGBoost with BMI feature engineering (Kaggle) | [View notebook](notebooks/en/Notebook_obesityRisk_en.ipynb) |
| **Real Estate Price Prediction** | Regression on property prices with text feature extraction and Optuna hyperparameter optimization (Kaggle) | [View notebook](notebooks/en/Notebook_realEstate_en.ipynb) |
